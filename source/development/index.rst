================================
Manuel du développeur de Galette
================================

Le projet Galette utilise la forge `Gna! <https://gna.org/>`_ pour son développement, afin de pouvoir utiliser les différentes possibilités qui sont offertes :

* `une page pour le projet Galette sur Gna! <https://gna.org/projects/galette/>`_,
* `un dépôt SVN pour gérer les source de galette <https://gna.org/svn/?group=galette>`_ ; dans lesquelles les modifications sont apportées,
* `un gestionnaire de bogues <https://gna.org/bugs/?group=galette>`_, pour nous signaler tout problème que vous auriez rencontré,
* `un gestionnaire de tâches <https://gna.org/task/?group=galette>`_, pour nous faire connaître les fonctionnalités ou autres améliorations que vous souhaiteriez voir intégrer dans le projet,
* `un gestionnaire de patches <https://gna.org/patch/?group=galette>`_, pour soumettre les modifications que vous auriez pu effectuer.

La présente documentation vous expliquera les règles de développement adoptées pour Galette, ainsi que la récupération du code source, etc.
